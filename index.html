Books and Materials

There's a lot of great books about computer science and linguistics that are worth reading. I'm going to list some important for me here or those I plan to read in the future. I also feel like reviewing them when I'm done, so I'll probably add a link to the reviews down here in the future.

  • Crafting Interpreters (2015-2021), Robert Nystrom — A didactic book about writing interpreters, lexers, and parsers. You build a programming language that features rich syntax, dynamic typing, garbage collection, lexical scope, first-class functions, closures, and inheritance in a series of chapters, while still maintaing a simple, clean and consistent code design.
  • Modern Compiler Implementation in ML (1997), Andrew W. Appel — A expanded textbook describing all phases of a modern compiler: lexical analysis, abstract syntax, semantic actions, intermediate representations, instruction selection via tree matching, dataflow analysis, graph-coloring register allocation, and runtime systems. It includes good coverage of current techniques in code generation and register allocation, as well as functional and object-oriented languages, that are missing from most books. In addition, more advanced chapters are now included so that it can be used as the basis for two-semester or graduate course.
  • [Portuguese] Linguagens Formais e Autômatos (2000), Paulo Fernando Blauth — A book about formal languages and automata theory.
  • The Stranger (1942), Albert Camus — A philosophical novel about the absurdity of life. Camus argues that the only certain thing in life is the inevitability of death.
  • Existentialism is a Humanism (2007), Jean-Paul Sartre — A philosophical essay by Jean-Paul Sartre, in which the author discusses the philosophy of existentialism and its relationship with humanism.
  • Being and Time (1927), Martin Heidegger — A book by Martin Heidegger, in which the author attempts to develop a philosophy of being that is free of the metaphysical presuppositions of traditional Western philosophy.

There's a lot of great learning material on the internet as well, which are essential knowledge for anyone interested in computer science, such as:

  • [Portuguese] Lógica proposicional — A playlist of videos teaching propositional logic.
  • An Introduction to Naive Set Theory, Cantor's Theorem, Russell's Paradox & the History of Set Theory, Amour Learning — An introduction to naive set theory
  • Math for Game Devs [2022], Freya Holmér — A playlist of videos teaching essential mathematics and physics for game developers.
